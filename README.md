# AI Power Quality Analyzer

An AI-powered Power Quality analysis tool that processes electrical measurement data,
detects anomalies, generates compliance reports, and visualizes trends through
interactive dashboards.

## Features

- **Multi-file Upload & Merge** — Upload multiple CSV/Excel files and merge them chronologically
- **Automatic Column Mapping** — Intelligent detection and normalization of PQ parameter columns
- **Interactive Dashboards** — Real-time charts for voltage, current, THD, power factor, and more
- **Anomaly Detection** — AI-based identification of sags, swells, harmonics, and transients
- **Compliance Reports** — Standards-based analysis (IEEE 519, EN 50160, IEC 61000)
- **PDF Export** — Generate professional reports with charts and findings
- **Analysis History** — View past analysis sessions and results
